Review of Vocabulary Words

Lesson Content: There is vs There are

There is

Use with singular nouns and uncountable nouns.

  • There is an apple.
  • There is some milk.
  • There is a lot of cheese.

There are

Use with plural countable nouns.

  • There are two clocks.
  • There are four strawberries.
  • There are many ice cubes.
